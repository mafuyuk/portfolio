# Portfolio
https://mafuyuk.com/

## 🌐 動作環境
* Node.js v10.11.0
* Vue.js v2.5.17

## ▶ 実行方法
### 環境立ち上げ
```bash
$ git clone git@github.com:mafuyuk/portfolio.git
$ npm install
$ npm run start
```

### storybook環境立ち上げ
```bash
$ npm run storybook
```

### デプロイ
```bash
$ npm run build
$ npm run deploy
```
